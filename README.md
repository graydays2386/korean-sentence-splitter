# korean-sentence-splitter

Rule-based candidate splitter for Korean informal compound sentences.
Designed as a preprocessing step before LLM-based refinement.

## Features
- Sentence boundary detection beyond punctuation
- Discourse marker–based splitting
- Morphology-aware connective ending heuristics
- Over-splitting–tolerant design for LLM post-processing

## Intended Use
- Knowledge management pipelines
- Wiki / document preprocessing
- Korean NLP experiments
